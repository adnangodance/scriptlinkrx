
  # Build responsive product UI

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Open the URL printed by Vite (normally `http://localhost:5173/Dashboard/`).
  Do not open `index.html` directly because the source entry point must be
  compiled by Vite.

  Run `npm run build` to create a production build, then `npm run preview` to
  preview that build locally.
